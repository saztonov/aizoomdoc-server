# Regression Scenarios (General Analysis Focus)

Цель: ручная проверка стабильности общего анализа и политики зумов.

## Scenario 1 — Общий анализ без зумов
- **Запрос:** «Оцени ключевые технические решения и предложи оптимизации по материалам и трассировкам».
- **Ожидание:** ответ строится на текстовых блоках (примечания/спецификации), без обязательных ROI.
- **Не должно быть:** подсчёта элементов как основного результата.

## Scenario 2 — Детали требуются, должен быть followup ROI
- **Запрос:** «Укажи диаметры и отметки на плане».
- **Ожидание:** `needs_more_evidence=true`, `followup_rois` запрашиваются.
- **Не должно быть:** точных значений без ROI‑цитат.

## Scenario 3 — Смешанный запрос (анализ + подсчёт)
- **Запрос:** «Дай обзор решений и примерно оцени количество ключевых элементов».
- **Ожидание:** общий анализ приоритетен; подсчёт краткий и только при наличии доказательств.

## Scenario 4 — Расхождение текст vs графика
- **Запрос:** «Сверь спецификацию с графикой и выяви несоответствия».
- **Ожидание:** при отсутствии ROI для графики — запрос followup ROI.

## Scenario 5 — Низкое качество превью
- **Запрос:** «Определи, что написано в таблице на плане».
- **Ожидание:** обязательный ROI запрос, без попыток «угадывать».
